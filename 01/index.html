HTML 標籤

HTML 的意義

  • HyperText Markup Language
  • Markup 為標籤的意思
  • 使用標籤 (Tag) 來標示功能單位，簡單易學、容易上手。

簡史

  1. HTML 2.0：1995年11月發布。
  2. HTML 3.2：1997年1月發布為 W3C 推薦標準 (W3C Recommendation)。
  3. HTML 4.0：1997年12月發布為 W3C 推薦標準。
  4. HTML 4.01 (小幅修正)：1999年12月發布為 W3C 推薦標準。
  5. HTML 5：2012年發布為 W3C 候選推薦。

Tag 標籤

  • 用小於 < 和大於 > 兩個符號所包住的文字，我們稱為標籤
  • 標籤通常成對存在，有開始標籤和結束標籤。
  • 夾在開始和結束標籤中的內容稱為標籤的內文 (inner HTML or inner Text)。
			<strong>我們要強調的文字，通常以粗體呈現</strong>
			<p>我們應該要<strong>快樂</strong>地學習</p>

超連結 a

利用<a>建立超連結。標籤屬性 href 的值代表要連結的網址，內文代表要顯示在畫面上的文字。

什麼是SEO？

  • 搜尋引擎優化(Search Engine Optimization)
  • 透過了解搜尋引擎的運作規則來調整網站

Permitted Contents

  • <p>, <h1>～<h6>標籤裡面只能包含 phrasing content
  • phrasing content 包含一般文字以及這些 tag
  • <ul>, <ol> 只能包含 <li>
  • <tr> 裡面只能包 <td>, <th>
  • 瀏覽器會容忍部分 HTML 錯誤

常見錯誤

  • <div>包含在<p>裡面
  • 怎麼知道行不行
			<p>
				<div>HI!</div>
			</p>
