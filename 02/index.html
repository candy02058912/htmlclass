id、class 如何選取？

  • id 用 # (取名時不要用數字開頭唷)
  • class 用 .
			#id名稱 {color: red;}
			.class名稱 {color: red;}

進階選取

			h2.blah {color: red;}
			有 blah class 之 h2
			h2, h3 {color: red;}
			h2和h3這兩個元素都設成字的顏色為紅色
			#blah p {color: red;}
			在 id 為 blah 之元素裡面的 p

CSS 版面配置

display

  • 每個 HTML 元素都有一個預設的 display 值
  • 通常是 block 或 inline 其中一個
  • block：區塊元素；inline：行內元素

display

  • <p>, <h1>~<h6>, <div> 預設 display: block;
  • <em>, <strong>, <span> 預設 display: inline;
  • display 還有 inline-block, none 的值
  • display: none 完全把該項目隱藏

inline vs inline-block

  • inline-block可以設 width 跟 height
  • inline 的 margin, padding 只能橫向發展
  • inline 的高要靠 line-height 設定

Box Model

  • 用來描述一個元素是如何組成的
  • DEMO

padding 和 margin 的寫法

  • 每一邊都加 15px 的 padding
    padding: 15px;
  • 只有上面的 padding 設為 15px
    padding-top: 15px;
  • padding 簡寫的順序為上右下左（順時針）
    padding: 14px 15px 16px 17px;

padding 和 margin 的寫法

  • 當有四個值的時候
    padding: 上 右 下 左;
  • 當有兩個值的時候
    padding: 上下 左右;
  • 當只有一個值的時候
    padding: 全部;

position 屬性

position: static | relative | absolute | fixed

  • static: 預設的定位模式 (由上到下、由左到右)
  • relative: 相對於預設的位置做位移
  • absolute: 絕對定位，直接指定標籤區快出現的座標位置
  • fixed: 相對於瀏覽器視窗來定位，即便頁面捲動還是會固定在相同的位置
  • z-index: 疊疊樂，數字大的在上面
