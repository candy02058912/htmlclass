選單 Navs

  • 基礎類別 .nav <div class="nav 樣式">
  • 樣式
    • .nav-tabs
    • .nav-pills
  • 選單是用 ul 清單來列出一個一個連結，Bootstrap 會把前面的點點拿掉
  • 清單裡面必須為超連結才有用
  • 在 li 加上 class="active" 可以標記該項目
  • 在 li 加上 class="disabled" 就會不能按下去
  • active / disabled 只要 class 裡面有 nav 就可以用

選單範例

See the Pen Navs by Candy (@stringpiggy) on CodePen.

巡覽列 Navbar

  • 這個部分比較複雜，主要是要會使用
  • 連結部分一定要是超連結
  • .navbar-brand 可以打上標題名稱
  • .navbar-right 把東西拉到巡覽列的最右邊
  • 可以用 .navbar-fixed-top 讓選單固定在上方，別忘了 body 要給 padding-top

嵌入內容 embed

  • 可以使用在 iframe, embed 和 object 元素上
  • 不用在 iframe 設定 frameborder="0"
  • 外層先用 div 包住 class="embed-responsive 尺寸"
  • 尺寸有 .embed-responsive-16by9 .embed-responsive-4by3

嵌入效果 well

  • 有大、中、小可以使用
  • .well-lg, .well-sm 加在原本的 well 後面 （使用兩個 class 的意思）
